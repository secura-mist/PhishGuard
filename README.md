# PhishGuard 🛡️

PhishGuard is a browser extension designed to help protect users from phishing websites. It analyzes visited URLs and warns users if a site appears suspicious.

## Features

* Detects suspicious URLs
* Displays warning page for phishing sites
* Lightweight browser extension
* Simple popup interface

## Project Structure

icons/ – extension icons
background.js – background service worker
content.js – content script for webpage analysis
manifest.json – extension configuration
popup.html – extension popup interface
popup.js – popup functionality
styles.css – styling for UI
warning.html – phishing warning page

## Installation

1. Download or clone this repository
2. Open Chrome
3. Go to `chrome://extensions`
4. Enable **Developer Mode**
5. Click **Load unpacked**
6. Select the project folder
